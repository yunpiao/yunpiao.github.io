python import mysql 出错

2014/03/03

升级系统后 不知道怎么回事 import mysql突然提示
ImportError: libmysqlclient.so.16: cannot open shared object file: No such file or directory
刚好这几天有事 这种感觉就像在挖坑

解决方法

首先查找自己有没有这个so

find / -name “libmysql*”

google 发现在/usr/lib/ 里创建这个软连接 可是无用 因为 我没有libmysqlclient.so.16文件

之后发现在/usr/lib64/mysql/libmysql* 有许多 18 的so

现在 假设 是升级 把我的16删了 升级到了18 :s
so :sad:

在同一文件夹下发现 libmysqlclient.so 根据理所应当效应 我觉得这个就是我的16

so

ln -s /usr/lib64/mysql/libmysqlclient.so /usr/lib64

so
import MySQLdb OK !!

2

sudo apt-get install libmysqlclient-dev
