dpkg 安装 deb 包 提示amd64 i386内核不符解决方法

2014/03/03

升级系统后提示 dpkg 安装 deb 包 提示amd64 i386内核不符解决方法

错误如下

dpkg: 处理归档 teamviewer_12.0.71510_i386.deb (–install)时出错：
软件包体系架构(i386)与本机系统体系结构(amd64)不符
在处理时有错误发生：
teamviewer_12.0.71510_i386.deb

apt 增加 i386 支持

原文如下：

Run Windows applications like MS Office
On 64-bit flavors of Debian and some Ubuntu variants, first open a terminal and run ‘sudo dpkg –add-architecture i386 ; sudo apt-get update’.
CrossOver Linux makes it possible to run Windows productivity applications such as Microsoft Office, and Quicken, and also lets you play Windows games like Skyrim and World of Warcraft.
