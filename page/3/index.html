python жҖ§иғҪеҲҶжһҗ cProfile

еҸ‘иЎЁдәҺ 2014/03/03

Pythonж ҮеҮҶеә“дёӯжҸҗдҫӣдәҶдёүз§Қз”ЁжқҘеҲҶжһҗзЁӢеәҸжҖ§иғҪзҡ„жЁЎеқ—пјҢеҲҶеҲ«жҳҜcProfile, profileе’ҢhotshotпјҢеҸҰеӨ–иҝҳжңүдёҖдёӘиҫ…еҠ©жЁЎеқ—statsгҖӮиҝҷдәӣжЁЎеқ—жҸҗдҫӣдәҶеҜ№PythonзЁӢеәҸзҡ„зЎ®е®ҡжҖ§еҲҶжһҗеҠҹиғҪпјҢеҗҢж—¶д№ҹжҸҗдҫӣдәҶзӣёеә”зҡ„жҠҘиЎЁз”ҹжҲҗе·Ҙе…·пјҢж–№дҫҝз”ЁжҲ·еҝ«йҖҹең°жЈҖжҹҘе’ҢеҲҶжһҗз»“жһңгҖӮ

иҝҷдёүдёӘжҖ§иғҪеҲҶжһҗжЁЎеқ—зҡ„д»Ӣз»ҚеҰӮдёӢпјҡ

cProfileпјҡеҹәдәҺlsprofзҡ„з”ЁCиҜӯиЁҖе®һзҺ°зҡ„жү©еұ•еә”з”ЁпјҢиҝҗиЎҢејҖй”ҖжҜ”иҫғеҗҲзҗҶпјҢйҖӮеҗҲеҲҶжһҗиҝҗиЎҢж—¶й—ҙиҫғй•ҝзҡ„зЁӢеәҸпјҢжҺЁиҚҗдҪҝз”ЁиҝҷдёӘжЁЎеқ—пјӣ

profileпјҡзәҜPythonе®һзҺ°зҡ„жҖ§иғҪеҲҶжһҗжЁЎеқ—пјҢжҺҘеҸЈе’ҢcProfileдёҖиҮҙгҖӮдҪҶеңЁеҲҶжһҗзЁӢеәҸж—¶еўһеҠ дәҶеҫҲеӨ§зҡ„иҝҗиЎҢејҖй”ҖгҖӮдёҚиҝҮпјҢеҰӮжһңдҪ жғіжү©еұ•profilerзҡ„еҠҹиғҪпјҢеҸҜд»ҘйҖҡиҝҮз»§жүҝиҝҷдёӘжЁЎеқ—е®һзҺ°пјӣ

hotshotпјҡдёҖдёӘиҜ•йӘҢжҖ§зҡ„CжЁЎеқ—пјҢеҮҸе°‘дәҶжҖ§иғҪеҲҶжһҗж—¶зҡ„иҝҗиЎҢејҖй”ҖпјҢдҪҶжҳҜйңҖиҰҒжӣҙй•ҝзҡ„ж•°жҚ®еҗҺеӨ„зҗҶзҡ„ж¬Ўж•°гҖӮзӣ®еүҚиҝҷдёӘжЁЎеқ—дёҚеҶҚиў«з»ҙжҠӨпјҢжңүеҸҜиғҪеңЁж–°зүҲжң¬дёӯиў«ејғз”ЁгҖӮ
